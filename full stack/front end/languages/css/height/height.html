CSS Height

What is CSS Height


the height property is used to set the height of an element. The height property specifies the height of the content area of an element. It can be applied to various HTML elements, such as divs, paragraphs, images, and more.

1.height: 100px; sets the height to 100 pixels.

2.height: 50%; sets the height to 50% of the parent element's height.

3.height: 2em; sets the height to twice the size of the current font size.

Example


<Doctype! html >
<html >
<head >
<title>.............. </title> </head >
<body>

<style>
h5{
background-color: red;
height: 200px;
}
</style>
<div>
<h5>iam height</h5>
</div>

</body>
</html>

Output




iam height

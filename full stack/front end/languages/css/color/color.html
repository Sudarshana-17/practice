CSS colors

What is CSS color


In Cascading Style Sheets (CSS), colors can be defined using various formats. The most common ways to specify colors in CSS are:

1.Keyword Names: CSS supports a set of predefined color names, such as red, blue, green, etc.

2.Hexadecimal Notation: Colors can be represented using a hexadecimal notation, which consists of a # followed by six characters (three sets of two characters) representing the RGB (Red, Green, Blue) values.

3.RGB Function: Colors can also be defined using the rgb() function, which takes three parameters representing the Red, Green, and Blue values in the range 0 to 255.

4.RGBA Function: Similar to rgb(), the rgba() function is used for specifying a color along with an alpha (transparency) value. The alpha value ranges from 0 (completely transparent) to 1 (completely opaque)

5.HSL Function: The hsl() function allows you to specify a color using Hue, Saturation, and Lightness values.

6.HSLA Function: Similar to hsl(), the hsla() function includes an additional alpha (transparency) parameter.

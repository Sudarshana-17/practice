HTML tag


What is HTML tag ?


The html tag is a fundamental element in HTML (Hypertext Markup Language) and serves as the root element of an HTML document. Every HTML document should begin with an opening html tag and end with a closing html tag. The entire content of the HTML document is contained between these tags.

The tag encapsulates the entire structure of the HTML document, providing a container for organizing and structuring content. It is an essential element in the hierarchy of HTML tags.

Example


<html >
.
.
.
</html >

HTML iframe Tag

What is iframe Tag ?


The HTML iframe (inline frame) tag is used to embed another HTML document within the current HTML document. It allows you to display content from another source, such as a web page or video, within the context of your own webpage. The iframe tag is often used to embed maps, videos, forms, or other external content.

The src attribute specifies the source URL of the content to be embedded. The width and height attributes define the dimensions of the iframe.

Example


<Doctype! html >
<html >
<head >
<title>.............. </title> </head >
<body>

<iframe width="400px" height="200px" src="https://getbootstrap.com/docs/5.3/components/list-group/" >

</body>
</html>

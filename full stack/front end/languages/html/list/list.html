HTML Lists

What is Lists ?


In HTML (Hypertext Markup Language), you can use the ul (unordered list) and ol (ordered list) elements to create lists. Within these elements, you can use li (list item) elements to define individual items in the list.

ol (ordered list) arranges the items in numeritical form

ul (unordered lis) arranges the items without using numeritical form

Example


<Doctype! html >
<html >
<head >
<title>.............. </title> </head >
<body>

<ol>

<li>list 1</li>
<li>list 2</li>
<li>list 3</li>

</ol>

</body>
</html>

Output




  1. list 1
  2. list 2
  3. list 3

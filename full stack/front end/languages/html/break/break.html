HTML Break Tag

What is Break Tag ?


In HTML, the <br> tag is used to insert a line break within text or elements. The name "br" stands for "break." When the browser encounters this tag, it causes the content following the tag to start on a new line. The br tag is an empty tag, meaning it doesn't have a closing tag.

Example


<Doctype! html >
<html >
<head >
<title>.............. </title> </head >
<body>

<p>hii this is <br> sudarshan </p>
</body>
</html>

Output




hii this is
sudarshan

HTML Hyperlink Tag

What is Hyperlink Tag ?


In HTML (Hypertext Markup Language), the a element is used to create hyperlinks, also known as anchor links or simply links. The "a" in a stands for "anchor." The a element allows you to define a hyperlink that can be used to link to another web page, a file, an email address, or any other URL.

The href attribute specifies the destination URL, which can be a relative or absolute URL.

The text between the opening <a> tag and the closing </a> tag is the clickable link text that appears on the webpage.

Example


<Doctype! html >
<html >
<head >
<title>.............. </title> </head >
<body>

<a href="https://chat.openai.com/c/4ce7901e-99ee-43f0-8a10-9a12e4601ad4" >chatgpt</a>

</body>
</html>

Output




chatgpt

HTML Video Tag

What is Video Tag ?


In HTML (Hypertext Markup Language), the video tag is used to embed video files or video streams on a web page. It allows you to include video content that can be played directly within the web browser. The video tag supports various attributes and can contain multiple source elements to provide different video formats for compatibility with different browsers.

width and height: These attributes specify the width and height of the video player on the web page.

controls: This attribute adds playback controls to the video player, such as play, pause, and volume control.

Example


<Doctype! html >
<html >
<head >
<title>.............. </title> </head >
<body>

<video src="VID20191018120942 - Copy.mp4" width="300px" height="300px" controls">

</body>
</html>

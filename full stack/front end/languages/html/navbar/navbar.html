HTML nav Tag

What is nav Tag ?


In HTML, the nav tag is used to define a navigation section on a web page. It is intended to contain links to other pages or to different sections within the same page, providing a way for users to navigate through the website. The nav element is part of the HTML5 specification and is semantically used to mark up the main navigation block of a document.

Example


<Doctype! html >
<html >
<head >
<title>.............. </title> </head >
<body>

<nav>

<a href="/full stack/front end/languages/html/image/image.html">image</a>
<a href="/full stack/front end/languages/html/videos/videos.html">video</a>
<a href="/full stack/front end/languages/html/heading/heading.html">heading</a>
<a href="/full stack/front end/languages/html/heading/title/title.htm">title</a>

</nav>

</body>
</html>

Output




image | video | heading | title

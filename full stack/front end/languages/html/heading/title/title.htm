HTML Title Tag

What is Title Tag ?



In HTML (Hypertext Markup Language), the title tag is used to define the title of a webpage. The title is typically displayed in the title bar of the browser or in the browser's tab. It is also used as the default name for bookmarking the page.

It's important to note that the title tag is placed within the head section of the HTML document, not in the body section. This is because it provides metadata about the document rather than content that is displayed on the webpage.

Example


<Doctype! html >
<html >
<head >
<title>.....title name......... </title>
</head >
<body>
.
.
.
</body>
</html>

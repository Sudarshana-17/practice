HTML Image Tag

What is Image Tag ?


The HTML img tag is used to embed an image in a web page

src: This attribute specifies the URL or file path of the image. It can be an absolute URL (starting with "http://" or "https://") or a relative URL (relative to the current HTML file).

alt: This attribute provides alternative text for the image. The text is displayed if the image cannot be loaded, and it is also used by screen readers for accessibility. It's considered good practice to provide descriptive and meaningful alternative text.

width and height set the dimensions of the image.

style allows you to apply inline CSS styles. In this case, a border is added around the image.

It's important to note that the img tag is self-closing, meaning there is no closing /img tag.

Example


<Doctype! html >
<html >
<head >
<title>.............. </title> </head >
<body>

<img src="image.jpg" alt="simple image" width="300px" height="300px" style="border: 1px solid black;">

</body>
</html>
